Resource Profile: Perfil de Localización de Objetos e Infraestructura

Defining URL: http://core.hl7chile.cl/StructureDefinition/CoreLocalizacionCl
Version: 0.85 (Ballot Pendiente)
Name: LocalizacionCL
Title: Perfil de Localización de Objetos e Infraestructura
Status: Active as of 2022-01-18T11:06:02-03:00
Definition:

Este perfil determina la ubicación especifica para un evento, objeto o Infraestructura específica. Este perfil permite vinlcular la localización especifica con una organización o espacio físico específico
Source Resource: XML / JSON / Turtle

The official URL for this profile is: 

http://core.hl7chile.cl/StructureDefinition/CoreLocalizacionCl
